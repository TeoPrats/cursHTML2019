What are this tags for?

- <br>	Representa un salto de línea.
- <hr>	Representa un quiebre temático entre parrafos de una sección o articulo o cualquier contenido.
- <ul>	Define una lista de artículos sin orden.
- <ol>	Define una lista ordenada de artículos.
- <li>	Define un artículo de una lista ennumerada.
- <strong>  Representa un texto especialmente importante  
- <b>	Representa un texto hacia el cual se llama la atención para propósitos utilitaros.
- <em> 	Representa un texto enfatizado
- <i>	Representa un texto en una voz o estado de ánimo alterno, o por lo menos de diferente calidad
- <u>	Representa una anotación no textual sin-articular, como etiquetar un texto como mal escrito
- <h1>	Los elemento de cabecera  implementan seis niveles de cabeceras de documentos; <h1> es la de mayor y <h6> es la de menor importancia
- <h6>	
- <tr>	Representa una fila de celdas en una tabla.
- <th>	Representa una celda encabezado en una tabla.
- <td>	Representa una celda de datos en una tabla.
- <sub>	Representa un subíndice.
- <sup>	Representa un superíndice.
- <legend>  Representa el título de un <fieldset>.
-<fieldset>  Representa un conjunto de controles.
